---

 Typography Design Guide (With `rem`)  

---

 Color Palette  
- Primary Color (Blue): `5271FF`  
- Secondary Accent Color (Dark Blue): `4051D3`  
- Neutral Text (Dark Gray): `333333`  
- Secondary Text (Light Gray): `666666`  
- Background (Light): `F9FAFB`  
- Background (Dark Elements): `000000`  
- Borders/Dividers: `DDDDDD`  

---

 1. Typography Rules  

 Headings (H1–H6)  
Font: Montserrat  

- H1 (Main Titles):  
  - Font Size: `4.8rem`  
  - Font Weight: `700`  
  - Line Height: `1.2`  
  - Letter Spacing: `-0.05rem`  
  - Color: Primary Blue (`5271FF`)  

- H2 (Section Titles):  
  - Font Size: `3.6rem`  
  - Font Weight: `600`  
  - Line Height: `1.3`  
  - Color: Dark Gray (`333333`) or Primary Blue (`5271FF`)  

- H3 (Subsection Titles):  
  - Font Size: `2.8rem`  
  - Font Weight: `500`  
  - Line Height: `1.4`  
  - Color: Dark Gray (`333333`)  

- H4–H6:  
  - H4: Font Size: `2.4rem`, Weight: `500`  
  - H5: Font Size: `2rem`, Weight: `500`  
  - H6: Font Size: `1.6rem`, Weight: `500`  
  - Line Height: `1.5`  
  - Color: Secondary Gray (`666666`)  

---

 Body Text (Paragraphs)  
Font: Roboto  

- Primary Paragraphs (P):  
  - Font Size: `1.6rem`  
  - Font Weight: `400`  
  - Line Height: `1.6`  
  - Color: Dark Gray (`333333`)  

- Secondary/Small Text (Meta, Labels):  
  - Font Size: `1.4rem`  
  - Font Weight: `400`  
  - Line Height: `1.4`  
  - Color: Light Gray (`666666`)  

---

 2. Links and Navigation  

- Main Navigation Links:  
  - Font: Montserrat  
  - Font Size: `1.6rem`  
  - Font Weight: `600`  
  - Letter Spacing: `0.05rem`  
  - Color: Primary Blue (`5271FF`)  
  - Hover Color: Dark Blue (`4051D3`)  

- Inline Text Links:  
  - Font: Roboto  
  - Font Size: `1.6rem`  
  - Font Weight: `400`  
  - Color: Primary Blue (`5271FF`)  

---

 3. Buttons  

- Primary Button:  
  - Font: Montserrat  
  - Font Size: `1.6rem`  
  - Font Weight: `600`  
  - Text Transform: Uppercase  
  - Background: Primary Blue (`5271FF`)  
  - Text Color: White (`FFFFFF`)  
  - Padding: `1.2rem 2.4rem`  
  - Border Radius: `0.8rem`  
  - Hover Background: Dark Blue (`4051D3`)  

- Secondary Button (Outlined):  
  - Font: Montserrat  
  - Font Size: `1.6rem`  
  - Font Weight: `500`  
  - Text Transform: Uppercase  
  - Text Color: Primary Blue (`5271FF`)  
  - Background: Transparent  
  - Border: `0.2rem solid 5271FF`  
  - Hover Background: Primary Blue, Text Color White  

---

 4. Forms and Input Fields  

- Labels:  
  - Font: Montserrat  
  - Font Size: `1.4rem`  
  - Font Weight: `600`  
  - Color: Light Gray (`666666`)  

- Input Fields:  
  - Font: Roboto  
  - Font Size: `1.6rem`  
  - Font Weight: `400`  
  - Padding: `1rem 1.5rem`  
  - Border: `0.1rem solid Light Gray` (`DDDDDD`)  
  - Focus State: Border Primary Blue, Background White  

---

 5. Footer Text  

- Footer Text:  
  - Font: Roboto  
  - Font Size: `1.4rem`  
  - Font Weight: `400`  
  - Color: Light Gray (`666666`)  
  - Line Height: `1.6`  

- Footer Links:  
  - Font: Montserrat  
  - Font Size: `1.4rem`  
  - Font Weight: `600`  
  - Color: White (`FFFFFF`)

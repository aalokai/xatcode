Usage Guidelines

Brand Elements

  • Use Brand Blue (#2563EB) as the primary color for logos and main CTAs
  • Innovation Purple (#7C3AED) for highlighting tech features
  • Deep Blue (#1E40AF) for hover states and emphasis

Typography

  • Text Black (#0F172A) for main headings and body text
  • Gray Dark (#334155) for secondary text
  • Gray Medium (#64748B) for supporting text

UI Elements

  • Success Green (#059669) for positive actions and success states
  • Error Red (#DC2626) for error messages and warnings
  • Gray Light (#E2E8F0) for borders and separators
